What does the project do?

In this project, the goal is to build a machine learning model that could classify the bank notes into two different categories. For thie project, we will use this dataset
In our dataset, are five different columns/attributes in our data file. Among them, 4 of them are our probable feature attributes (Image.Var, Image.Skew, Image.Curt, Entropy), while 1 of them is our target variable (Class). Since, we have both target and feature variables, this problem can be expressed as supervised machine learning classification problem. All the feature variables are continuos, while target variable is categorical.

Machine Learning Models


Logistic Regression Classifier: Explanation

Logistic Regression is a classification technique used in machine learning. It uses a logistic function to model the dependent variable. The dependent variable is dichotomous in nature, i.e. there could only be two possible classes ( 0, or 1) in our case.

Logistic Regression: Results

Here we have a confusion matrix with the results of the logistic regression applied on our testing dataset. A confusion matrix represents the prediction summary in matrix form. It shows how many prediction are correct and incorrect per class. It helps in understanding the classes that are being confused by model as other class. In the figure below, we can see that our logistic regression model misclassifies 3 Class 1 data as Class 0, and 23 Class 0 data as Class 1.

Useful Evaluation Metrics:
Accuracy=0.98
F1-Score=0.98
AUC=1

Neural Networks: Explanation

The main purpose of a neural network is to try to find the relationship between features in a data set., and it consists of a set of algorithms that mimic the work of the human brain. A “neuron” in a neural network is a mathematical function that collects and classifies information according to a specific architecture. Neural networks are one of the most powerful machine learning approaches for data with complicated patterns, but are more prone to overfitting to simple data.

Neural Networks: Results

Looking at the confusion matrix below, the neural network performed much better on classifying Class 0 bank notes.

SVM Classifier: Explanation

Support vector machines (SVMs) are a set of supervised learning methods used for classification, regression and outliers detection. The objective of the support vector machine algorithm is to find a hyperplane in an N-dimensional space(N — the number of features) that distinctly classifies the data points. Support vector machine is highly preferred by many as it produces significant accuracy with less computation power.

SVM Classifier: Results

Looking at our confusion matrix below, SVM Classifier performed well on classifying both classes correctly compared to other two model used above.

Useful Evaluation Metrics:

AUC=1

Conclusion


In conclusion, SVM classifier is the best model on our dataset for classifying bank notes as it outperforms other model in terms of accuracy and F1-score. I am using F1-score as another evaluation metric because our dataset is not perfectly balanced. So, in such scenario, F1-score is the better metric to evaluate model performance.
